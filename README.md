# ELowMP
Basically a desktop application with the idea of not having to open google to listen to music from YouTube and SoundCloud.

## Status 
- [ ] Front-end
- [ ] Play Youtube
- [ ] List Youtube search
- [ ] Play SoundCloud
- [ ] List SoundCloud search
- [ ] Play Local Files
- [ ] Play FM Radio
